Javascript কিভাবে কাজ করে?

Javascript একটি প্রোগ্রামিং ভাষা । ওয়েভসাইটের ক্লায়েন্ট এর কাজে ব্রাউজার এর বিহেভিয়ার লক্ষ্য করে এটি ডিজাইন করা হয়েছে। javascript HTML এবং css এর সাথে অনেক ভালো কাজ করে তাই এটির ব্যবহার এবং জনপ্রিয়তা দিন দিন বেড়েই চলেছে। এটি ব্রাউজার ছাড়া ও রান করা যায় এবং এটি হাই লেবেলের ল্যাঙ্গুয়েজের মতো কাজ করে। javascript অনেক ডেভলাপারের পছন্দের একটি ল্যাঙ্গুয়েজের। এটি ব্রাউজারের সাথে সহজেই কাজ করা তাই এটি দিয়ে কাজ করা সহজ।

LocalStorage এবং SessionStorage মধ্যে পার্থক্য |

LocalStorage হলো এটির ডাটা ব্যবহারকারী নিজে ডিলিট না করা পর্যন্ত ডেটা বজায় থাকে। SessionStorage হলো এটির ডাটা ব্যবহারকারী ট্যাব বন্ধ না করা পর্যন্ত ডেটা বজায় থাকে। এ দুটিতে প্রায় একই api থাকে। LocalStorage এ আগে থেকেই কিছু ডাটা থাকে যা ব্যবহারকারী নিজে ডিলিট না করলে LocalStorage এ থেকে যায়। SessionStorage ব্যবহারকারীর কিছু ডাটা add করা হয় এবং ট্যাব বন্ধ করার সাথে সাথে ডিলিট হয়ে যায়।
